# DSC 530 – Data Exploration and Analysis

This repository contains coursework completed as part of the Master of Science in Data Science
program at Bellevue University.

**Course:** DSC 530 – Data Exploration and Analysis  
**Institution:** Bellevue University  

The assignments in this repository focus on introductory data exploration tasks using Python
and pandas, including data cleaning, reshaping, and exploratory analysis.

Textbook and instructor-provided materials are not included.
